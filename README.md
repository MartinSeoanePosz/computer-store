# Computer Store

> Tienda de items de computacion


Tienda con varios items de computacion, separados por categorias y un carrito funcional con form y requerimientos

## Built With

- ReactJs - Jsx



## Como usarlo

**Se arranca con el comando "npm run dev" en la terminal**
